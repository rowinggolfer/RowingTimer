NOTES:
included files.
1. index_style.css is a stylesheet for the indexpage, called for all media except "print"
2. index_print_style.css is a stylesheet should anyone try to print the index page (it basically hides everything except advice on how to print the results you want.)
3. ruler.js a javascript file to highlight any row with a mouseover.
4. printit.js is a javascript file to enable printing.
